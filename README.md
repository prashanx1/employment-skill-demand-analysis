# Employment & Skill Demand Analysis

## Overview
This project analyzes real-world job posting data to understand employment trends,
high-demand skills, and location-wise job distribution for Data Analyst roles.
The goal is to derive insights that can support large-scale employment and skilling initiatives.

## Problem Statement
With growing demand for data-driven decision-making, it is essential to understand
which skills and locations dominate the job market. This analysis helps identify
skill gaps and employment patterns relevant to digital employment platforms like ONEST.

## Dataset
- Source: Public job posting dataset
- Records: Data Analyst job listings
- Key fields: Job Title, Company, Location, Salary, Job Description

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Analysis Performed
- Data cleaning and standardization
- Location-wise job demand analysis
- Job title distribution
- Skill demand extraction from job descriptions

## Key Insights
- Python, SQL, and Excel are the most in-demand skills
- Metro cities dominate Data Analyst job postings
- Data Analyst and Business Analyst roles appear most frequently
